## คอลัมน์ที่ข้าม (ไม่ import)

- A, B → ชื่อ/รหัสโรงเรียน (ไป SETTINGS)
- N, O → อายุ (คำนวณจาก birth_date)
- Z, AF, AM → เลขบัตร พ่อ/แม่/ผู้ปกครอง (PDPA — ไม่จำเป็น)
- T–X, CF–CK → จำนวนพี่น้อง (ไม่ใช้ใน 6 เมนู)
- AA–AK → ข้อมูลบิดามารดาละเอียด (ใช้แค่ผู้ปกครองหลัก)
- AS–BB → ที่อยู่ตามทะเบียนบ้าน (ใช้ที่อยู่ปัจจุบันพอ)
- BM–BY → ด้อยโอกาส/พิการ/ระยะทาง/ประเภท (ไม่ใช้เฟสแรก)
- BZ, CA → GPA/GPAX (ระบบวัดผลแยก)
- CB–CE → จังหวัดเกิด/อาชีพ (ไม่ใช้เฟสแรก)
- CL, CM → โรงเรียนสาขา (ไม่มี)
